# Fun Website for Friends

এটি একটি মজার ওয়েবসাইট যা React দিয়ে তৈরি করা হয়েছে।

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- ✅ Responsive design for Android and iPhone
- ✅ Bengali/Bangla text support
- ✅ Modern and beautiful UI
- ✅ Mobile-friendly layout
