# Fun Website for Friends

এটি একটি মজার ওয়েবসাইট যা React দিয়ে তৈরি করা হয়েছে।

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

### Render.com

এই প্রজেক্টটি Render.com-এ deploy করার জন্য প্রস্তুত। `render.yaml` file আছে যা automatic deployment configure করে।

**Deployment Steps:**
1. Render.com-এ নতুন Static Site তৈরি করুন
2. GitHub repository connect করুন
3. Build Command: `npm install && npm run build`
4. Publish Directory: `dist`
